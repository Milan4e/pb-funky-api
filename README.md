# pb-funky-api

# npm install

# npm start

## Service will run on localhost:3000


## API Docs

### Create board

POST /api/feature/boards

BODY
{
  "name": "My first board"
}

### Get all boards

GET /api/feature/boards

### Create widget

POST /api/feature/boards/o9J_lZ6I-3Y=/widget (where o9J_lZ6I-3Y= is board ID)

BODY
{
  "name": "My first board"
}
